# Generate Random Networks

Generate Random Networks

## Usage

``` r
generate_random_networks(n, e, reps = 1000, output_file = NULL)
```

## Arguments

- n:

  Number of nodes.

- e:

  Number of edges.

- reps:

  Number of random networks to generate, default 1000.

- output_file:

  Output file path for summary (optional).
